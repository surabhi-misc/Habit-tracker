# 🟢 Group Habit Tracker

A **collaborative habit tracking app** that helps groups stay accountable to shared goals. Track daily habits, see progress, and celebrate streaks together!

## ✨ Features

- 🎯 **Shared Goal** - Set a common goal for your group
- 📋 **Flexible Tasks** - Add multiple tasks/habits to track
- 👥 **Member Management** - Invite group members and assign them tasks
- ✅ **Weekly Check-ins** - Track completion for each day of the week
- 🔥 **Streak Tracking** - Show off multi-day streaks
- 📊 **Progress Metrics** - See daily completion rates and overall progress
- 📲 **WhatsApp Integration** - Share check-ins directly to WhatsApp
- 🌙 **Dark Mode** - Automatic light/dark theme support
- 💾 **Offline-First** - All data saved locally; works without internet

## 📖 How to Use

### 1. **Set Your Goal** ✏️
Click "Edit" on the goal card to customize:
- Pick an emoji
- Name your goal (e.g., "Morning Routine", "Fitness Challenge")
- Add a subtitle

### 2. **Add Tasks** ➕
Click "+ Add task" to create habits:
- Assign an emoji
- Name the task
- Select which members opt into it

### 3. **Add Members** 👥
Click "+ Add member":
- Enter their name
- Select which tasks they're doing

### 4. **Check In Daily** ✅
Each member clicks the day buttons to mark tasks as complete:
- **Green checkmark** = Completed
- **Letter (M/T/W...)** = Not completed
- **Disabled** = Future day (can't check in ahead)

### 5. **Share Progress** 📲
Use the "Post check-in to WhatsApp" button to send an emoji-formatted summary to your group.

## 🎨 Customization

Everything is customizable through the UI:
- Change the goal emoji and description
- Add/remove tasks and members
- Manage who's opted into what
- All changes save automatically to your browser

## 💾 Data Storage

- **Local Storage**: Data is saved in your browser (IndexedDB + localStorage)
- **No Cloud Required**: Works 100% offline
- **No Account Needed**: Just share the URL
- **Shared Access**: Anyone with the link can edit and see progress
